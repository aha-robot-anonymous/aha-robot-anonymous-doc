Welcome to AhaRobot's documentation!
====================================

**AhaRobot** is a low-cost mobile robotics platform designed to lower the entry barrier in the field of Embodied AI, enabling more researchers and users to participate.

This documentation will guide you through the complete process of building, configuring, and using your own AhaRobot system.

.. note::

   This project is under active development.
